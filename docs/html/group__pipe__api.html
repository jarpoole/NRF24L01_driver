Writes a TX payload to the FIFO and triggers a transmission.

Parameters
[in]data: Pointer to a buffer with the TX payload data, remember the TX FIFO is 32 bytes long
[in]len: Length of buffer to transmit (in bytes)
[in]platform: Driver instance configuration struct pointer
Return values

NRF24L01_ERR_UNKNOWN-> Device communication failed
NRF24L01_OK-> Payload write was successful
